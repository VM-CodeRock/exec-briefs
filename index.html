Karpathy's AutoResearch: The Autonomous ML Research Loop

Open Source • MIT License

Andrej Karpathy — former Tesla AI lead, OpenAI co-founder, and coiner of "vibe coding" — released autoresearch, a minimalist 630-line Python script that automates the scientific method for machine learning. The agent reads its own source code, forms hypotheses, modifies training parameters, runs experiments within a fixed compute budget (typically 5 minutes on a GPU), evaluates results, and iterates — all without human involvement.

126 Experiments Overnight
~700 Changes in 2 Days
11% Efficiency Gain
8.6M+ Views on X

In one overnight run, the agent drove validation loss from 0.9979 to 0.9697. Over two days of continuous operation, it processed ~700 autonomous changes, finding ~20 additive improvements that transferred to larger models — cutting the "Time to GPT-2" leaderboard metric from 2.02h to 1.80h. Karpathy noted the agent caught attention scaling and regularization oversights he had missed in two decades of manual work.

Viral adoption and distributed scaling: Varun Mathur (CEO, Hyperspace AI) distributed the loop across a peer-to-peer network. On the night of March 8–9, 35 autonomous agents ran 333 experiments unsupervised. Key findings:

  • Hardware diversity as strategy — H100 GPUs used brute force while CPU-only agents got creative, focusing on initialization and normalization strategies
  • Gossip-based discovery — When one agent found Kaiming initialization dropped loss by 21%, the discovery spread network-wide within hours via GossipSub protocol; 23 agents incorporated it
  • Compressed history — In 17 hours, agents independently rediscovered ML milestones (RMSNorm, tied embeddings) that took human researchers at Google Brain and OpenAI ~8 years to formalize

Strategic implication: This demonstrates AI-driven research loops that are applicable far beyond ML — any domain involving hypothesis → experiment → evaluate cycles (marketing, drug discovery, materials science) could benefit from this paradigm.

Microsoft 365 E7: The $99 "Frontier Suite" for Agentic Enterprise

Enterprise • GA May 1, 2026

Microsoft announced Microsoft 365 E7, its highest-tier enterprise subscription at $99 per user per month, available May 1, 2026. Dubbed the "First Frontier Suite built on Intelligence + Trust," E7 bundles the full E5 stack with AI agent management and advanced security into a single license.

$99 Per User / Month
13.2% Bundle Discount
$15 Agent 365 Standalone
May 1 GA Date

What's included in the bundle:

  • Microsoft 365 E5 — Full existing top-tier productivity and compliance suite
  • Microsoft 365 Copilot — AI assistant across Office, Teams, and Outlook
  • Agent 365 — New control plane for governing AI agents (access controls, monitoring, lifecycle management)
  • Microsoft Entra Suite — Identity and access management
  • Advanced Defender, Intune & Purview — Enhanced security, device management, and data governance for both humans and AI agents

"Gartner believes organizations will find the value of ME7 to be questionable for the majority of knowledge workers today... Upgrading to the ME7 bundle for Agent 365 is not advised until Microsoft adds value."

— Gartner Analysis, March 2026

Analyst caution: Gartner found the 13.2% bundle discount underwhelming compared to historical E3→E5 discounts, called Agent 365 "a work in progress with limited net new functionality," and warned enterprises to avoid nonreduction clauses in case Microsoft revises the offerings due to low uptake.

Strategic implication: Microsoft is positioning for an "agentic enterprise" future where AI agents are licensed like employees. The market signal is clear — but the product may be ahead of enterprise readiness. Wait-and-see advised unless already deep in the Microsoft AI ecosystem.

Anthropic Code Review: Multi-Agent Bug Detection for Claude Code

DevTools • Research Preview

Anthropic launched Code Review, a multi-agent system integrated into Claude Code that automatically analyzes pull requests for logic bugs before human reviewers see the code. Available in research preview for Claude for Teams and Claude for Enterprise customers.

3× Review Feedback Gain
$2.5B+ Claude Code Run-Rate
4× Enterprise Sub Growth YTD

The problem: Claude Code has dramatically increased code output, creating a bottleneck at the PR review stage. As Cat Wu (Anthropic's Head of Product) explained: "Now that Claude Code is putting up a bunch of pull requests, how do I make sure those get reviewed in an efficient manner?"

How it works:

  • Integrates with GitHub; auto-triggered on PR creation
  • Multiple agents analyze the codebase in parallel from different perspectives
  • A final aggregation agent deduplicates, ranks, and prioritizes findings
  • Focuses strictly on logic errors over style — high-signal, actionable feedback only
  • Step-by-step reasoning explains what the issue is, why it's problematic, and how to fix it
  • Engineering leads can customize additional checks based on internal best practices

Severity classification:

Critical — must fix
Potential — worth reviewing
Historical — preexisting issues

Enterprise context: Target customers include Uber, Salesforce, and Accenture. Claude Code's run-rate revenue has surpassed $2.5B since launch. Anthropic is leaning hard into enterprise dev tooling, especially as it navigates a legal dispute with the Department of Defense over a supply chain risk designation.

Strategic implication: As AI-generated code volume explodes, automated quality assurance becomes critical infrastructure. Anthropic is building the full pipeline — code generation (Claude Code) → code review (Code Review) → security analysis (Claude Code Security) — creating significant switching costs for enterprise customers.
