SharePoint CSV Consolidation Agent

Automated data pipeline — daily CSV drops to normalized Tableau-ready database

Automation Data Pipeline SharePoint Implementation Plan
The Problem: Daily CSV file drops land in dated subfolders on SharePoint. Today, someone manually consolidates them into a single dataset for Tableau. This agent watches for new drops, waits 30 minutes for all files to arrive, then auto-merges them into a single normalized CSV — zero data modification, zero manual effort.

2. Watcher Logic

The agent polls the SharePoint folder at a configurable interval (default: every 5 minutes). When it detects a new dated subfolder that hasn't been processed:

  1. First file detected — starts a 30-minute countdown timer
  2. Timer running — continues monitoring for additional files
  3. 30 minutes elapsed — triggers consolidation
  4. Processing — reads all CSVs, validates schemas, concatenates
  5. Output — writes consolidated-YYYY-MM-DD.csv to output folder
⚠️ Data Integrity Rule: No modifications to source data. The agent reads and concatenates only. Column names preserved exactly as-is. Row order preserved within each file.

3. Consolidation Rules

  • Schema alignment: Union of all columns across files. Missing columns filled with empty values.
  • Source tracking: Adds _source_file column with original filename for traceability
  • Deduplication: None — preserves all rows exactly as received
  • Encoding: Auto-detects UTF-8 / Latin-1; outputs UTF-8 with BOM for Excel compatibility
  • Delimiter: Auto-detects (comma, tab, pipe, semicolon)

Core Script: csv-consolidator.py

#!/usr/bin/env python3
"""
SharePoint CSV Consolidation Agent
- Watches for new dated subfolders
- Waits 30 min after first file detected
- Consolidates all CSVs into single normalized output
- Zero data modification
"""

import os, glob, json, time, csv, chardet
from datetime import datetime, timedelta
from pathlib import Path

CONFIG = {
    "watch_dir": "/path/to/synced/sharepoint/folder",
    "output_dir": "/path/to/output",
    "state_file": "/path/to/state.json",
    "delay_minutes": 30,
    "poll_interval_seconds": 300,  # 5 minutes
}

def detect_encoding(file_path):
    with open(file_path, 'rb') as f:
        result = chardet.detect(f.read(10000))
    return result['encoding'] or 'utf-8'

def detect_delimiter(file_path, encoding):
    with open(file_path, 'r', encoding=encoding) as f:
        sample = f.read(8192)
    sniffer = csv.Sniffer()
    try:
        return sniffer.sniff(sample).delimiter
    except csv.Error:
        return ','

def consolidate_folder(folder_path, output_path):
    """Read all CSVs in folder, union columns, concat rows."""
    csv_files = sorted(glob.glob(os.path.join(folder_path, "*.csv")))
    if not csv_files:
        return 0

    all_rows = []
    all_columns = []

    for csv_file in csv_files:
        enc = detect_encoding(csv_file)
        delim = detect_delimiter(csv_file, enc)
        with open(csv_file, 'r', encoding=enc) as f:
            reader = csv.DictReader(f, delimiter=delim)
            for col in reader.fieldnames or []:
                if col not in all_columns:
                    all_columns.append(col)
            for row in reader:
                row['_source_file'] = os.path.basename(csv_file)
                all_rows.append(row)

    all_columns.append('_source_file')

    with open(output_path, 'w', encoding='utf-8-sig', newline='') as f:
        writer = csv.DictWriter(f, fieldnames=all_columns,
                                extrasaction='ignore')
        writer.writeheader()
        for row in all_rows:
            writer.writerow({col: row.get(col, '') for col in all_columns})

    return len(all_rows)

def main():
    """Main watch loop."""
    state = load_state()
    while True:
        dated_folders = discover_new_folders(state)
        for folder_name, folder_path in dated_folders:
            if folder_name not in state.get('pending', {}):
                state.setdefault('pending', {})[folder_name] = {
                    'first_seen': datetime.now().isoformat(),
                    'path': folder_path
                }
                save_state(state)

        # Check pending folders past the 30-min mark
        for folder_name, info in list(state.get('pending', {}).items()):
            first_seen = datetime.fromisoformat(info['first_seen'])
            if datetime.now() - first_seen >= timedelta(
                    minutes=CONFIG['delay_minutes']):
                output_file = os.path.join(
                    CONFIG['output_dir'],
                    f'consolidated-{folder_name}.csv')
                rows = consolidate_folder(info['path'], output_file)
                state.setdefault('completed', {})[folder_name] = {
                    'rows': rows,
                    'completed_at': datetime.now().isoformat()
                }
                del state['pending'][folder_name]
                save_state(state)

        time.sleep(CONFIG['poll_interval_seconds'])
